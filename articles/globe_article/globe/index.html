Transparent WebGL Canvas Demo

This text sits behind the 3D globe. If you can read this through the empty space around the globe, the WebGL canvas transparency is working correctly — just like a PNG with an alpha channel.

The globe's dark sphere and glowing dots should remain fully opaque, while the canvas background is completely see-through.

How It Works

  • Bevy's ClearColor is set to fully transparent RGBA (0, 0, 0, 0)
  • The WebGL2 context preserves its alpha channel
  • CSS layers the canvas above this text using z-index
  • The canvas container has pointer-events: none so clicks pass through

Features

The globe renders 32,000 land dots using a Fibonacci sphere distribution, filtered against a land mask texture. Animated arcs pulse between highlighted cities (Paris, New York, Tokyo). The camera auto-rotates and supports click-and-drag orbit control.
