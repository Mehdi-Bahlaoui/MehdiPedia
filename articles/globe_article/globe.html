Interactive Globe

Cool Globe isn't it? It was made using Bevy, a Rust game engine, and compiled to WebAssembly to run in the browser.

You can click and drag to rotate the globe.

This was a failed project unfortunately as I lost motivation to finish a very big project that I had planned to build on top of this globe, but I still wanted to share it as I think it's pretty cool on its own!
