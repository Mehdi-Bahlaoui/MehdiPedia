Privacy Policy

1. Privacy Statement

This Privacy Policy documents UDPCore's approach to user privacy.
UDPCore facilitates direct UDP communication with ESP devices, prioritizing minimal latency while maintaining complete user privacy through a zero-external-data architecture.

2. Data Collection Policy

2.1 What We Do Not Collect

UDPCore does not collect, store externally, or transmit:

  • Personal identification information
  • Device IP addresses or network configurations to external servers
  • UDP command contents or sensor data to external services
  • Usage analytics, telemetry, or behavioral data
  • Location data or device fingerprints
  • Any form of tracking or profiling data

2.2 Local Data Handling

UDPCore includes the following local components:

  • Settings Page: Saves user interface preferences locally on device
  • About Page: Static informational content explaining UDP concepts
  • UDP Interface: Real-time communication bridge - no data logging

Note: Settings preferences are stored locally using your device's standard storage mechanisms and never transmitted externally.

3. Internet Permission Usage

UDPCore requires internet permission exclusively for:

  • UDP Transmission: Sending control commands to ESP devices on your network
  • UDP Reception: Receiving responses and data from ESP devices
  • Network Socket Management: Managing real-time UDP connections

Limitation: Internet permission is used solely for local network UDP communication. No external servers, APIs, or third-party services are contacted.

4. Future Development

Planned UDPCore enhancements include:

  • Reception labels for ESP sensor data display
  • Enhanced UI controls (additional buttons, sliders)
  • Improved device management interfaces

Commitment: All future features will maintain the zero external data collection architecture. Real-time UDP communication remains the sole network function.

5. Minors' Privacy

UDPCore's zero external data collection model inherently protects users of all ages. No age verification or data collection occurs for any user demographic.

6. Policy Updates

Privacy Policy modifications will be communicated through application updates. Continued use indicates acceptance of updated terms. Our zero external data commitment remains unchanged.
